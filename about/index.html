这里是 Derived Cat ~ 感谢关注我的网站

2016 年至今，我大部分笔记都使用 $\LaTeX$ 排版，分享在 Github 上，主要包括数学、机器学习、经济学。一方面为了学习 WebGL，另一方面想记录一些不那么正式的笔记和不太成熟的想法，我也开始使用个人博客了 ~

本站的内容主要涉及数学与应用数学、人工智能与机器学习、计算机图形。文章不追求深度，只希望具有原创性。互联网世界潜藏着无数珠玉，同时有用信息的密度也非常之低。我唯一的目标就是多多少少写出一点新的东西、提供一些新的角度，成为其他学习者的帮助而不是干扰 ~

除了文章，本站还在 Demo 页提供了 WebGL 示例。可右键直接查看源码，也可在 Github 上查看相关代码。WebGL 作为轻量级的图形 API，很适合充当概念验证与论文复现的试验场。当然，它也能平滑现代图形 API 陡峭的学习曲线 ~

感谢前人开发的 Hexo 框架和 Fluid 主题，让像我一样不熟悉 Web 技术的人，也能快速发布自己的网站 ~
